# Changelog

All notable repository-level changes to the comon_logger monorepo workflow,
documentation, and release process are documented here.

## 0.1.2

- Rebuilt the repository as a Dart workspace with Melos configuration living in the root `pubspec.yaml`.
- Moved publishable packages under `packages/` and aligned root docs, repository links, and development commands with the new monorepo layout.
- Added shared GitHub issue templates, PR template, CI and publish workflows, plus repository-level contribution, security, and conduct documents.
- Cleaned analyzer output across the workspace and fixed logger package validation issues uncovered during the migration.
